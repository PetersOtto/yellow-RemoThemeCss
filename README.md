# yellow-RemoThemeCss

If you develop your own theme and use your own folder structure, the `theme.css` file from Yellow may be superfluous.
This Yellow extension removes the theme CSS link from the HTML source code.

Please take a look to this discussion:
https://github.com/datenstrom/community/discussions/1060
